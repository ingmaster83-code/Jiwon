# 지원넷 프로젝트 지침

## 프로젝트 개요
- **사이트명:** 지원넷 (JiwonNet)
- **URL:** https://jiwon.wooahouse.com
- **GitHub:** https://github.com/ingmaster83-code/jiwon
- **배포:** GitHub Pages (main 브랜치 → root)

## 기술 스택
- 순수 HTML / CSS / JS (프레임워크 없음)
- PWA: manifest.json + sw.js + js/pwa-install.js

## 서비스 목적
국가지원금·정부지원금 정보를 분야별로 안내하는 비공식 정보 포털.

## 페이지 목록
- index.html — 전체 허브
- youth.html — 청년 지원금
- housing.html — 주거 지원
- childcare.html — 임신·출산·육아
- welfare.html — 기초생활·복지
- employment.html — 취업·고용
- elderly.html — 노인 지원
- education.html — 교육·학자금
- tax-credit.html — 근로·자녀장려금
- energy.html — 에너지·생활비
- startup.html — 창업·소상공인
- health.html — 의료·건강
- disability.html — 장애인 지원
- about.html, privacy.html

## 작업 규칙
- 새 지원금 추가 시 sitemap.xml 업데이트
- 모든 외부 링크는 공식 정부 사이트만 사용
- 비공식 사이트 안내 문구 반드시 유지
- SEO 키워드: 국가지원금, 정부지원금, 청년지원금, 기초연금, 근로장려금
